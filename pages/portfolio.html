  • NextTrain

    This was a project which required the use of Firebase and moment.js to build an application to keep track of train schedules. The application can be used to add trains to the Firebase database and to display all key information about each. The application also uses moment.js to calculate upcoming arrival times and the number of minutes from now until the next train.

  • EmojiCat

    Get a cat gif to express your mood with one click! Simple app that gets gifs from the Giphy API with a cat theme- you can click any button and it will search for that word plus "cat". You can also add additional buttons with a form, and toggle the gifs between still and animated by clicking. The layout uses bootstrap to make it responsive.
